## CLI

```
SYNOPSIS
       <URL> [OPTIONS]

OPTIONS
       -a, --algorithm   SHA-n, one of: 1, 256 (default), 384, 512.
       -f, --format      base64 (default) or hex.
       -m, --max-time    fetch maximum time in seconds (default 10).
       -p, --prefix      output the algorithm by prepending (default false).
```
